Notes & Limitations

Scope: This calculator produces MWFRS (Main Wind Force Resisting System) pressures per ASCE 7-22 Directional Procedure (Chapter 27). These values are used for design of the lateral load resisting system (frames, shear walls, diaphragms, connections).

Not included: Components & Cladding (C&C) loads (Chapter 30), which govern design of individual windows, doors, roofing panels, and fasteners. C&C pressures are higher and require separate analysis using GCp coefficients from ASCE 7-22 Figures 30.3-1 through 30.3-7.

Assumptions: Enclosed building (GCpi = ±0.18); rigid structure (G = 0.85); Exposure Category B (suburban/urban); flat terrain (Kzt = 1.0); sea level (Ke = 1.0); square plan (L/B = 1.0, leeward Cp = −0.5); roof slope ≤ 10°. Boston basic wind speed V = 115 mph per ASCE 7-22 Table 26.5-1, Risk Category II (700-yr MRI).
